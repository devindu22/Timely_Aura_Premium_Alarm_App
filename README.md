# 🌌 Timely Aura

An ultra-premium, interactive, glassmorphic recreation and modernization of the legendary **Timely Alarm Clock** application. Built entirely with vanilla modern web technologies, heavy-duty CSS glassmorphism, mathematical canvas fluid styling, and native browser Web Audio synthesis.

---

## ✨ Key Features

### 1. 🎭 The Interactive Aura Splash Screen
- **Fluid Visual Loader:** On boot, an organic, multi-layered rotating SVG infinity/hourglass emblem pulses against fluid gradient flares.
- **Dynamic Progress Syncing:** Features a custom loading bar that handles viewport asset initialization before smoothly fading visibility.

### 2. 🧪 Aura Lab & Liquid Shader Themes
- **Mathematical Moving Canvas Engine:** Powered by a customized Canvas 2D engine executing independent vector velocity equations to simulate floating liquid metaballs/ambient aura blobs.
- **Palette Presets with Live Visual Dots:** Choose between **Midnight Indigo**, **Emerald Aurora**, **Sunset Fusion**, and **Cyberpunk Neon**. Each theme box provides high-fidelity glowing physical color indicators showcasing the current layout shade matrix.

### 3. ⏰ Advanced Alarms & Custom Dropper Menus
- **Tactile Physical Slide Buttons:** The alarm toggle switches utilize high-contrast absolute positioning layouts, smooth CSS transforms, and rich physical track colors (`bg-slate-800` to gradient neon tracks) matching premium smartphone operating system designs.
- **Glassmorphic AM/PM Selector Dropdown:** Replaced rigid standard text boxes with a beautiful, fully customized glass backdrop drop list that opens with fluid translation transitions.
- **Precision 1-Minute Selection Tracks:** Built-in side-ruler drag listeners allowing micro-precision scrolling to dial in exact times by the minute, enhanced with velocity-based spring stretching ("gel/rubber-band" physics).
- **Enriched Smart Rise Panel:** Features extended inner margins and layouts to present the 30-minute pre-alarm ambient chime feature elegantly.

### 4. ⏳ Aura Chronos (Timer & Stopwatch)
- **Mechanical Roller Tumbler Wheels:** Cylinder-modeled vertical scrolling numbers (Hours, Minutes, Seconds) supporting touch swipe drag physics, mouse scroll wheels, and bounding threshold calculations accompanied by simulated physical click ticks.
- **Adaptive Running Interface Layout:** When a timer countdown initializes, all editing controls (`-1m`, Reset, `+1m`) instantly slide out of sight, transforming the workspace to focus entirely on full-width **Cancel** and **Play/Pause** interaction buttons.
- **Custom Reset-to-Preset Symbol:** Integrated a beautiful, clean circular undo/reset SVG icon to gracefully return your active dial back to your previously selected time matrix.
- **Dedicated Expiry Overlays:** Upon timer completion, a specialized full-screen alert hides standard alarm "Snooze" triggers, rendering an explicit pulsing layout customized for timers.
- **Stopwatch Precision:** Accompanied by lap logging capabilities that save records chronologically in an interactive dashboard list.

### 5. 🌙 Bedtime & Real Sleep Analytics
- **Web Audio API Ambient Synthesis:** Purely procedural soundscape generation (Cozy Rain, Rolling Ocean Waves, Deep Space Drone). The Ocean mode tracks active Low-Frequency Oscillators (LFOs) to modulate a multi-stage noise envelope for authentic breathing wave simulations—no heavy external media files required.
- **Persistent Local Database:** Tracks, computes, and stores genuine historical logs (total hours slept, verification puzzle solve latency).
- **Live SVG Vector Charts:** Translates sleep matrix metrics into beautiful, translucent multi-shaded bar graphs rendering live trends in the browser.

### 6. 🧠 Wakeup Verification Challenges
Prevents oversleeping by forcing engagement with custom puzzles before the Web Audio pentatonic chime can be fully dismissed:
- **Math Arithmetic:** Generates random synchronized equations with integrated shake-error visual warning boxes for wrong outputs.
- **Memory Tile Match:** A randomized matrix requiring matching symbol pairs.
- **Rapid Burst Tap:** A decaying progress meter requiring rapid screen taps to exceed a 100% burst threshold.

## 🛠️ Technical Stack & Architecture

- **Structure:** Semantic HTML5 Structure
- **Styling:** Tailwind CSS + High-End Advanced Glassmorphic Filter Overlays (`backdrop-blur-3xl`, Saturation multipliers, `box-shadow` depth rings)
- **Typography:** Google Fonts (Outfit & Inter)
- **Core Engine:** Vanilla JavaScript (ES6 Modules/Event System Architecture)
- **Audio Engine:** Native Web Audio API Synthesizers (`AudioContext`, `OscillatorNode`, `GainNode`, `BiquadFilterNode`, custom noise buffers)
- **Data Layer:** Persistent Web `localStorage` Pipeline

---

## 🚀 Getting Started

No compilation, installations, or external dependencies are required! Because **Timely Aura** is built natively, it runs directly in any modern mobile or desktop web browser.

1. Clone or download the repository files.
2. Locate `index.html`.
3. Open `index.html` directly in Google Chrome, Safari, Microsoft Edge, or Mozilla Firefox.
4. For the absolute best user experience on mobile, add the application to your smartphone home screen as a Progressive Web App (PWA) layout or run it via a local development server!
